# Satelight-image

Urban change detection project with a Python backend and a Vite + React frontend.

## Structure
- `backend/` API and server code
- `frontend/` UI (Vite + React)
- `models/` trained model artifacts (ignored)
- `outputs/` inference outputs (ignored)
- `data/` datasets (ignored)

## Quick start

### Backend
```powershell
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```powershell
cd frontend
npm install
npm run dev
```

## Notes
- Large artifacts and datasets are excluded via `.gitignore`.
